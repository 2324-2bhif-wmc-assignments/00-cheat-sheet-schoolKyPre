text commands

  • color: Ändert die Textfarbe.
  • font-family: Ändert die Schriftart.
  • font-size: Ändert die Schriftgröße.
  • font-weight: Macht den Text fett.
  • font-style: Setzt den Text in kursiv.
  • text-decoration: Fügt eine Textdekoration hinzu (z.B., Unterstreichung).
  • text-align: Ändert die Textausrichtung.
  • white-space: Steuert den Zeilenumbruch und Leerzeichen.
  • text-transform: Ändert die Textdarstellung (z.B., in Großbuchstaben).
  • line-height: Legt den Zeilenabstand fest.
  • text-shadow: Fügt einen Schatten zum Text hinzu.
  • background-color: Ändert die Hintergrundfarbe des Texts.
  • direction: Ändert die Textausrichtung von links nach rechts.
  • word-wrap: Kontrolliert den Zeilenumbruch von Wörtern.
  • absolute length Units
  • cm => centimeters
  • mm => millimeters
  • q => quarter-millimeters
  • in => inches
  • pc => picas
  • pt => points
  • px => pixels

    • why? printer

  • ---------------------------------------------
  • relative length Units
  • rem => font size of root element
  • ex => x height of the elements font

background commands

  • background: linear-gradient(); Erzeugt einen linearen Farbverlauf als Hintergrund.
  • background-color: Ändert die Hintergrundfarbe des Elements.
  • background-image: Legt ein Hintergrundbild fest.
  • background-repeat: Steuert, wie das Hintergrundbild wiederholt wird (z.B. no-repeat, repeat-x).
  • background-size: Legt die Größe des Hintergrundbildes fest (z.B. cover, contain).
  • background-position: Setzt die Position des Hintergrundbildes (z.B. center, left top).
  • background-attachment: Steuert, ob das Hintergrundbild beim Scrollen fixiert bleibt.
  • background: Eine Abkürzung, um alle Hintergrund-Eigenschaften in einer Zeile festzulegen.

other commands

  • float: right; lässt das element nach rechts "schweben"
  • margin-top: 10px; von element außerhalb von border weg 10 px
  • padding: 10px; von element nach innen oben weg 10 px
  • border: 3px solid aqua fügt eine 3px solide border in der farbe aque ein

animation commands

  • @keyframes fadeIn: Fades an element in by changing its opacity from 0 to 1.
  • @keyframes slideIn: Slides an element in from the left by translating it horizontally.
  • @keyframes rotate: Rotates an element 360 degrees.
  • @keyframes scale: Scales an element from 0 to 1, making it grow.
