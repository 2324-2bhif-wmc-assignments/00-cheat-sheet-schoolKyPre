Hyper Text Markup Language

basics
==============================

HTML Elemente

  • HTML-Element Ein HTML-Element erstreckt sich von der start-tag bis zum end-tag. <p>Ich bin ein Element.</p>
  • Void-Element Ein Tag wie <input> ist ein Void-Element - es hat keinen end-tag

Attribute

HTML-Elemente können Attribute haben, um zusätzliche Informationen über die Elemente bereitzustellen. Diese Attribute werden im Starttag angegeben und sehen normalerweise so aus: attributName="attributWert".

Überschriften

Überschriften werden mit den Tags <h1> bis <h6> definiert. Die Überschriften von 1 bis 6 werden immer weniger wichtig, wobei <h1> die wichtigste ist. Überschriften werden von Suchmaschinen verwendet, um den Inhalt und die Struktur Ihrer Webseiten zu indizieren.

Absätze

Ein neuer Absatz wird mit dem <p>-Tag definiert. Absätze beginnen immer auf einer neuen Zeile mit etwas Abstand davor und danach. Der Browser entfernt überflüssige Leerzeichen und Zeilen in einem Absatz, wenn die Seite angezeigt wird.

Es gibt auch zwei leere Tags, die in oder zwischen Absätzen verwendet werden: <br> und <hr>.

Der <br>-Tag definiert einen Zeilenumbruch (Zeilenumbruch), sogar in einem Absatz.

Der <hr>-Tag definiert einen thematischen Abschnitt auf einer HTML-Seite und wird meistens als horizontale Linie dargestellt. Sein häufigster Verwendungszweck ist die Trennung von Inhalten.

Kommentare

Kommentare können mit <!-- Das hier ist ein Kommentar --> dieser Syntax hinzugefügt werden. Das Ausrufezeichen (!) wird nur im Starttag verwendet. Kommentare können verwendet werden, um Inhalte zu verbergen und können über mehrere Zeilen erstreckt werden.

Links

Das <a>-Tag definiert einen Hyperlink. Das wichtigste Attribut von <a> ist href, das das Ziel des Links angibt. Wenn Sie auf die Links klicken, werden Sie zur angegebenen URL gesendet. Ein weiteres Attribut von <a> ist target. Mögliche Werte für das target-Attribut sind: _self, _blank, _parent, _top. Sie können fast alles zu einem Link machen, indem Sie <a> verwenden.

Bilder

Bilder können das Design und das Erscheinungsbild einer Webseite verbessern. Das <img>-Tag wird verwendet, um ein Bild einzubetten. Ein Bild wird nicht eingefügt, sondern mit der Webseite verknüpft. Dieses Tag ist ein leeres Tag und enthält nur Attribute. Wenn eine Webseite geladen wird, ist es der Browser, der das Bild von einem Webserver abruft und in die Seite einfügt.

Die beiden erforderlichen Attribute sind src (gibt den Pfad zum Bild an) und alt (gibt einen alternativen Text für das Bild an). Bilder können auch das Attribut height und width haben, um die Höhe und Breite des Bildes zu ändern (dies sollte jedoch in Ihrem Stylesheet erfolgen).

<head>

Das <head>-Element ist ein Container für die Elemente <title>, <style>, <meta>, <link>, <script> und <base>. Insgesamt ist das <head>-Element ein Container für Metadaten (Daten über Daten). Es befindet sich zwischen dem Starttag von <html> und dem Starttag von <body>. Die Daten aus dem <head>-Element werden nicht angezeigt, sondern nur vom Browser für Informationen verwendet.
